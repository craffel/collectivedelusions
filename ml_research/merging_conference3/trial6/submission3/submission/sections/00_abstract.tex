\begin{abstract}
While static model merging methods combine specialized expert networks into a single multi-task model at zero additional inference cost, they remain fundamentally task-agnostic. Dynamic model merging has emerged to address this by predicting sample-dependent routing coefficients at runtime. However, existing dynamic methods either learn unshared parameters for each of the $L$ layers (suffering from severe parameter scaling excess, high computational overhead, and cascading representation drift on small calibration splits) or rely on complex, non-monotonic wave-superposition routing that exhibits high sensitivity and optimization ruggedness across random seeds. True to an empirical philosophy of exhaustive validation, we deconstruct these issues and introduce the \textbf{Block-wise Weight-Sharing Router} (\textbf{BWS-Router}). BWS-Router groups the $L$ layers of the model into $G = L / M$ uniform blocks and shares routing weights within each block, reducing parameter complexity, slashing routing forward pass overhead by up to 91.7\%, and mathematically mitigating layer-to-layer coefficient ruggedness. Combined with an unsupervised PCA state pre-projector and bounded independent sigmoidal gating, BWS-Router is extremely parameter-efficient and stable. Through an exhaustive grid sweep of over $1,280$ experiment configurations across 5 independent seeds on our challenging task-conflict benchmark (MNIST, FashionMNIST, CIFAR-10, SVHN), we demonstrate that BWS-Router ($M=3$) achieves an outstanding \textbf{79.57 $\pm$ 1.14\%} Joint Mean accuracy (climbing to \textbf{79.63 $\pm$ 1.18\%} under optimal learning rates) using only 80 parameters, representing a 66.7\% parameter reduction while dramatically outperforming the static uniform baseline which completely collapses to \textbf{23.56 $\pm$ 2.91\%} under severe weight-space semantic conflicts. Furthermore, BWS-Router shows exceptional robustness under task-heterogeneous batch shifts and resolves out-of-distribution (OOD) performance collapse on noisy datasets.
\end{abstract}
