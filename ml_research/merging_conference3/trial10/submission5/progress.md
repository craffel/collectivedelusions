To maintain the ensembling weights on the probability simplex, they either force unnatural physical constraints (conservation of mass constraint $\kappa = k_{\text{decay}}$ in ChemMerge) or rely on post-hoc Softmax normalization over unconstrained states, introducing mathematical instability, representation barriers, and "representational lag" under sudden task transitions.

### Brainstorming 10 Novel Research Ideas
1. **Entropy-modulated Active Inference Routing (EMAIR):**
   - *Description:* Formulate routing as a 2D space-time active inference process where the bottom-up routing precision is dynamically modulated by the Shannon entropy of the local similarity signals, suppressing lag under confident switches.
   - *Expected Results:* Drastic reduction in jitter and lag, matching oracle accuracy under heterogeneous streams.
   
2. **Unitary Geodesic Routing (UGR):**
   - *Description:* A quantum-inspired stateful router operating on the unit sphere $\mathbb{S}^{K-1}$, where state updates are represented as exact orthogonal rotations generated by a skew-symmetric "torque" matrix. This guarantees exact conservation of ensembling probability without requiring a Softmax layer and provides smooth, geodesic trajectory filtering.
   - *Expected Results:* Complete eradication of routing jitter, zero scale-induced instabilities, and natural mitigation of lag via torque magnitude scaling.

3. **Cellular Automata Spatial Gating (CASG):**
   - *Description:* Treat the routing weights across deep layers and attention heads as cells in a 2D cellular automaton that evolve according to soft continuous transition rules based on neighboring routing decisions.
   - *Expected Results:* Emergence of self-organizing routing patterns that automatically group network layers into specialized task segments.

4. **Closed-loop PID-Controller Routing (PID-Route):**
   - *Description:* Model representation drift across network depth as a control error and regulate the ensembling trajectory using a Proportional-Integral-Derivative feedback controller.
   - *Expected Results:* Zero steady-state representation lag and robust noise damping.

5. **Event-driven Spiking Neural Router (SNN-Route):**
   - *Description:* Model the router as a Spiking Neural Network where activation features are encoded into spike trains and experts are selected based on firing synchrony.
   - *Expected Results:* Ultra-low serving latency and biological-like temporal noise immunity.

6. **Variational Active Inference on Task Manifolds (AIF-Merge):**
   - *Description:* Learn an amortized variational generative model of the task stream and update ensembling weights by minimizing variational free energy.
   - *Expected Results:* Mathematically optimal, bayesian-robust routing under extreme stream non-stationarity.

7. **Thermodynamic Diffusion Routing (TDR):**
   - *Description:* Model routing weight trajectories as a physical diffusion process governed by Langevin stochastic differential equations, where representation noise acts as a thermal bath.
   - *Expected Results:* Robust probabilistic ensembling with thermodynamic stability bounds.

8. **Graph Co-routing message-passing (GNN-Route):**
   - *Description:* Formulate the deep layers and expert adapters as nodes in a graph and perform message passing to compute mutually consistent routing weights.
   - *Expected Results:* Superior ensembling performance on highly overlapping representation manifolds.

9. **Adaptive Resonance Theory Routing (ART-Route):**
   - *Description:* Apply Grossberg's ART to dynamic model serving. The router matches inputs to expert expectation templates; a vigilance parameter triggers a search if a mismatch occurs.
   - *Expected Results:* Self-calibrating task boundary detection with absolute immunity to out-of-distribution drift.

10. **Neural Attractor Network Routing (NANR):**
    - *Description:* Formulate the routing state dynamics as a continuous Hopfield attractor network where task experts correspond to stable attractor basins.
    - *Expected Results:* High noise immunity by forcing the state to converge into clean, specialized expert basins.
