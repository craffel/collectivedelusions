# Evaluation: Novelty and Originality

## Key Novel Aspects
1. **The Transition to Multi-Tenant Stateful Routing:** Prior stateful routing works (like ChemMerge and PAC-Kinetics) are severely restricted by the assumption of continuous, single-user streams. Under interleaved streams, their temporal smoothing mechanism causes "state contamination." The concept of **Tenant-Decoupled Stateful Routing (TDSR)** is a highly original and timely response to this major gap between theory and practical deployment.
2. **Implicit Tagless Clustering via Fixed Orthogonal Task Detectors:** Unsupervised clustering in routing is notoriously prone to clustering collapse (runaway slot attraction) and centroid drift. The authors propose an elegant conceptual solution: initialize and fix the centroids as orthogonal basis vectors matching the expert subspaces. This eliminates drift and collapse, and mathematically simplifies a complex cosine-similarity argmax to a sub-nanosecond **coordinate-argmax integer lookup**, bridging high-level mathematical modeling and register-level execution.
3. **The Slot-Tenant-Task Triad (Virtual Task Caching):** In the implicit tagless mode, the router decouples slot memory from individual user sessions ($\mathcal{O}(M)$) by mapping queries to a tiny, fixed pool of $K$ task-specific virtual slots (where $K \ll M$). This creates a shared, mutually beneficial, task-smoothed context cache across concurrent tenants without state cross-talk. This is an exceptionally elegant architectural design pattern that allows a multi-tenant cloud server to scale to infinite concurrency with a tiny, constant memory footprint.
4. **Dual-Clock Decay Policy:** The dual-clock decay formulation (combining logical session-step decay to prevent sequence washout with a secondary physical wall-clock timer for memory reclamation) resolves the trade-off between sequence isolation and resource garbage collection in high-throughput infrastructures.
5. **Intra-Session Jitter Metric:** The paper identifies that traditional global jitter measurements are conceptually flawed under interleaved streams (where rapid context switches are *necessary* and *intended*), and introduces intra-session jitter as the correct metric to measure the true temporal smoothing capability of a router.

## The 'Delta' from Prior Work
- **Delta from SABLE (Stateless):** While SABLE dynamically routes queries based on activation projections, it is stateless and highly sensitive to sample-level noise. TDSR adds recurrent stateful smoothing *without* introducing the state contamination that a global stateful router would suffer from.
- **Delta from ChemMerge & PAC-Kinetics (Global Stateful):** These methods maintain a single global state vector $\mathbf{s}_t$, which works well on clean sequential streams but completely collapses under interleaved multi-tenant streams. TDSR's "delta" is the virtual state slot decoupling mechanism (Slot-Kinetics), which isolates temporal updates across different tenants or task contexts.
- **Delta from Multi-Tenant PEFT serving frameworks (Punica, S-LoRA, vLLM):** These frameworks solve low-level GPU memory tiling, batch scheduling, and KV-cache block allocation, but are agnostic to recurrent routing state dynamics. TDSR operates orthogally at the routing scheduler level, cleanly separating the routing state lifecycle from physical KV-cache page lifecycles.

## Characterization of Novelty
The novelty of this submission is **highly significant and paradigm-shifting** for the field of test-time model merging:
- It is **not** a minor, incremental tweak or hyperparameter tuning of existing stateful routers.
- Instead, it introduces a major conceptual leap: **how to maintain and track recurrent, low-pass-filtered routing memory across massive multi-tenant systems without suffering from state contamination or database/network overhead.**
- By identifying and formalizing the state contamination bottleneck and proposing Slot-Kinetics, the paper provides a crucial conceptual framework that allows stateful ensembling to finally transition from a theoretical single-user abstraction to practical, real-world multi-user cloud gateways.
- The mathematical simplification of cosine-similarity clustering into coordinate-argmax assignment is a stellar example of an elegant conceptual insight that leads to highly practical system efficiency.
- This paper has the potential to fundamentally change how the community thinks about deploying dynamic, stateful multi-expert models in high-throughput cloud and edge infrastructures.
