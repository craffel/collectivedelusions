This paper introduces \textbf{Winner-Take-All Sign Election (WTA-Sign)}, a minimalist, training-free, and hyperparameter-free method for model merging. The core idea is to resolve destructive interference in weight space (sign conflicts) by leveraging parameter update magnitude as a proxy for "task confidence." For each parameter, WTA-Sign identifies the expert model with the largest absolute parameter update. This "winning" expert's update direction dictates the sign for the merged parameter. Other experts' updates that conform to this sign are averaged, while conflicting updates are masked out. The authors argue this approach is simpler and superior to existing multi-stage heuristic methods like TIES-Merging, which involve pruning, sign-voting, and rescaling.

Empirical validation is presented on three vision tasks (MNIST, SVHN, CIFAR10) using an OpenCLIP ViT-B-32 backbone. The paper claims WTA-Sign consistently outperforms Task Arithmetic and TIES-Merging, particularly in avoiding performance collapse across different scaling coefficients ($\lambda$). Specifically, WTA-Sign is stated to maintain an average accuracy of 14.19\% (retaining zero-shot base capability) while being entirely hyperparameter-free. The implementation is highlighted as exceptionally elegant, requiring only four lines of vectorized PyTorch code.
