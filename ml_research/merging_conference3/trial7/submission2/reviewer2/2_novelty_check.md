# 2. Novelty Check

## Key Novel Aspects
1. **Dynamic test-time coordinate warping via Fisher Information:** While Fisher Information has been extensively used for static, offline weight ensembling (e.g., Fisher Merging) or to prevent catastrophic forgetting (e.g., Elastic Weight Consolidation), this work is the first to utilize diagonal empirical Fisher Information (dFIM) as a dynamic, on-the-fly metric tensor to warp representation spaces at test-time. It provides a formal connection showing that representation-space Fisher coordinates are mathematically equivalent to inverse coordinate-wise noise variances under conditional Gaussian assumptions.
2. **Mitigations for Practical Routing Biases:**
   - **Class-Size Scaling Calibration (CSC / CC-CSC):** Resolves the statistical maximum bias that arises when comparing similarity scores across tasks with asymmetric class vocabulary dimensions using extreme value theory.
   - **Micro-Batch Homogenization (MBH):** An unsupervised batch-partitioning mechanism that groups incoming heterogeneous samples into homogeneous micro-batches to allow a single forward pass per micro-batch, avoiding "heterogeneity collapse" without requiring $B$ separate forward passes for a batch of size $B$.

## Delta from Prior Work
- **From Static Merging (e.g., TIES-Merging, DARE, Task Arithmetic):** These methods compute a single, fixed set of merged weights. They are completely non-adaptive at test-time. FIOSR, by contrast, is fully dynamic and sample-adaptive, allowing the merged model to adjust its weights on the fly based on the specific domain of each incoming sample.
- **From Parametric Dynamic Routers (e.g., FoldMerge, QWS-Merge, L3-Softmax):** These frameworks rely on test-time optimization of routing parameters over calibration splits. This makes them highly susceptible to "The Dynamic Routing Paradox" (severe overfitting on tiny calibration sets) and "Vectorization Collapse" (extreme coefficient instability under sequential $B=1$ streams). FIOSR completely bypasses test-time parameter optimization, achieving zero-shot routing coefficient derivation that is immune to overfitting and sequential stream instability.
- **From Parameter-Free Subspace Routing (PFSR):** PFSR is the closest direct predecessor. It uses raw, unweighted cosine similarity to project representations onto frozen classification head weights. This implicitly assumes a flat, isotropic weight space. The critical "delta" of FIOSR is replacing the isotropic tensor $\mathbf{g} = \mathbf{I}$ with the regularized diagonal Fisher tensor $\mathbf{g} = \text{diag}(\tilde{F})$, which suppresses noisy dimensions and amplifies highly discriminative dimensions. This simple algebraic modification yields a substantial performance improvement ($+8.56\%$ in the synthetic sandbox, $+16.67\%$ routing accuracy in LoRA simulations).

## Characterization of Novelty
The novelty is **significant and highly creative**, but represents an **evolutionary step** rather than a complete paradigm shift. 
- **The Core Mechanics are Incremental:** The transition from PFSR's unweighted cosine similarity to a weighted cosine similarity using inverse-variance coordinate scaling is a classic statistical technique (equivalent to Mahalanobis distance with a diagonal covariance matrix). The use of dFIM to represent coordinate importance is well-established in other deep learning subfields.
- **The Concept Integration is Novel:** Unifying these concepts into a training-free dynamic test-time model merging framework is highly original and addresses a very pressing challenge in modular AI deployment. By proving that classifier weights act as dual vectors aligning with class centroids under $L_2$ regularization, the authors successfully ground the use of classification weights as proxies for activation-space centroids. This mathematical linking represents a substantial theoretical contribution.
- **Practical Engineering Utility:** From a practitioner's perspective, the true novelty is the combination of these elegant mathematical concepts with concrete engineering safeguards (CSC and MBH) to make the framework deployable under non-stationary streams and asymmetric class vocabularies. This makes the work far more practically relevant than purely theoretical model merging papers.
