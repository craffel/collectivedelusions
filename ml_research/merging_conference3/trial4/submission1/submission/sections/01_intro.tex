\section{Introduction}
\label{sec:intro}

The dominant paradigm in deep learning views neural network optimization as the navigation of high-dimensional real-valued Euclidean spaces. When fine-tuning a pre-trained backbone on various downstream tasks, this process yields a family of specialized, disjoint expert models. Rather than relying on the prohibitive computational cost of joint multi-task training from scratch, \emph{model merging} has emerged as a highly attractive, post-hoc alternative \cite{ilharco2022editing, wortsman2022model, yadav2023ties}. By directly blending the parameter matrices of downstream experts, model merging aims to construct a single unified model capable of performing all tasks simultaneously.

Yet, despite its practical appeal, current weight-space merging is bound by a fundamental limitation: it operates under the assumption that parameter space is flat, isotropic, and static. Traditional approaches like Task Arithmetic \cite{ilharco2022editing} or Model Soups \cite{wortsman2022model} merge expert task vectors ($\tau_k = W_k - W_{\text{pre}}$) via linear interpolation or simple averaging. In practice, this static real-space averaging triggers severe \emph{destructive parameter interference} (task conflicts), where updating a weight to satisfy one task actively degrades its performance on another.

To resolve task conflicts, recent work has proposed adaptive test-time model merging \cite{yang2023adamerging, regcalmerge2024}. By optimizing layer-wise linear coefficients $\Lambda$ over a small unsupervised calibration stream, these methods seek a customized parameter compromise. However, this optimization-centric path introduces the \textbf{Overfitting-Optimizer Paradox}: when calibration data is highly scarce (e.g., $M \le 16$), unconstrained unregularized layer-wise coefficients overfit to transductive validation noise. Rather than learning generalizable multi-task features, the optimizer constructs noisy, jagged parameter profiles that collapse catastrophically on unseen test data.

This fragility is amplified under the constraints of modern hardware deployment, which necessitates post-training quantization (PTQ) to low-bit depths (e.g., 8-bit or 4-bit) \cite{frantar2022gptq, lin2023awq}. When model merging parameters are optimized under a specific quantization schema, the learned configurations align themselves tightly with the discrete rounding boundaries of that exact operator \cite{qmerge2024}. If the deployment schema shifts---such as calibrating under an 8-bit schema but deploying on 4-bit edge devices---the optimized parameters can perform poorly. This \emph{target schema shift} highlights the extreme sensitivity of static real-space weight representations to high-frequency quantization noise.

In this work, we propose a novel paradigm: \textbf{PhaseMerge}, a frequency-domain model merging framework that projects fine-tuning updates into complex-valued Fourier space. Rather than treating parameter updates as static points, we represent task vectors in Fourier space:
\begin{equation}
    \mathcal{F}_k = \text{RFFT2D}(\tau_k) = A_k e^{i \theta_k}
\end{equation}
By projecting the parameters into complex-valued space, we decouple each frequency component into its magnitude $A_k$ and phase angle $\theta_k$. Because downstream expert networks are fine-tuned from a shared, pre-trained initialization backbone, their weights share a structurally aligned coordinate space. Operating within this shared basis, we explore two phase-rotation parameterizations that act as frequency-domain matrix-basis regularizers:
\begin{enumerate}
    \item \textbf{Uniform Phase Rotation ($r=1$), representing U-PhaseMerge:} We learn a single uniform phase-shift per layer and task (196 parameters total, which is highly comparable in dimensionality to AdaMerging's 192 parameters). By adjusting the global phase angle per layer and task, it serves as a highly structured, low-dimensional phase regularizer.
    \item \textbf{Continuous Phase-Shift Grids ($r=2$), representing PhaseMerge:} We learn a compact $2\times 2$ phase-shift grid per layer (772 parameters total), which is bilinearly upsampled to the full dimensions of the frequency tensor. This explores spatially smooth phase adjustments across different frequency scales.
\end{enumerate}
To preserve real-valued signal symmetry, we constrain both configurations with a strict symmetry-preserving mask on the DC and Nyquist components. Crucially, these phase-rotation mechanisms enable:
\begin{itemize}
    \item \textbf{Phase-Cancellation:} Conflicting updates or quantization noise can be rotated out of phase, actively mitigating task interference.
    \item \textbf{Coherent Reinforcement:} Aligned, synergistic task features can be adjusted in-phase to reinforce each other.
\end{itemize}

Crucially, the benefit of PhaseMerge lies not in reducing parameter count below that of AdaMerging, but in introducing a **frequency-domain matrix-basis regularizer**. By restricting phase-shifts to preserve conjugate symmetry, we enforce a physical coordinate constraint on the parameter Updates. This prevents the optimizer from fitting to sharp local rounding thresholds, providing excellent generalizability across diverse target post-training quantization schemas and stable multi-task convergence under moderate calibration sizes ($M \ge 16$).

\textbf{Our Contributions:}
\begin{itemize}
    \item We challenge the standard static Euclidean representation of neural network weights and propose \textbf{PhaseMerge}, a framework to model parameter merging via complex-valued wave superposition and phase rotation in the Fourier frequency domain.
    \item We design a highly structured **Uniform PhaseMerge (U-PhaseMerge, $r=1$)** formulation. This compact configuration (196 parameters) operates in a shared coordinate basis and exhibits outstanding robustness to extreme 4-bit quantization, outperforming traditional real-space layer-wise optimizations.
    \item We introduce a symmetry-preserving frequency mask that restricts phase adjustments on DC and Nyquist components to zero, mathematically securing real-valued spatial reconstructions and stabilizing optimization.
    \item Through extensive empirical evaluation on Vision Transformers (`vit\_tiny\_patch16\_224`) across 4 conflicting classification tasks, we demonstrate that PhaseMerge significantly outperforms static frequency-filtering baselines and achieves highly robust, quantization-resilient multi-task performance.
\end{itemize}
