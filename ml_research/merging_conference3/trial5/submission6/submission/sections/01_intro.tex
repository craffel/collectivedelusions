\section{Introduction}
\label{sec:intro}

The deployment of deep learning models across diverse, real-world multi-task applications is heavily constrained by hardware budgets and latency requirements. In production, a system must frequently handle streaming inputs spanning distinct tasks or visual domains. While deploying individual, fine-tuned expert models yields optimal performance, it incurs a prohibitive linear scaling cost in storage and memory, as maintaining $K$ separate copies of a modern model quickly exhausts edge device storage \cite{wortsman2022model, yang2023adamerging}. 

To mitigate this, weight-space \emph{model merging} has emerged as an attractive, training-free paradigm to consolidate multiple task-specific expert networks into a single multi-task model \cite{ilharco2022editing, yadav2023ties}. Early weight-merging approaches, such as Uniform Merging and Task Arithmetic \cite{ilharco2022editing}, rely on static, flat linear combinations of expert weights. While static merging requires no additional computational overhead at inference time, it suffers from severe \emph{task interference}, where conflicting parameter updates across different tasks degrade overall accuracy. 

To overcome this, recent advances have proposed \emph{dynamic model merging} (e.g., AdaMerging \cite{yang2023adamerging}, QWS-Merge, and Linear Routers), which dynamically adjust weight merging coefficients based on the input. By reconstructing the dense weights of the network on the fly using input-dependent scaling coefficients, these methods attempt to dynamically select the most relevant expert capacity.

However, existing dynamic merging methods suffer from a critical limitation: \textbf{batch-dependency} and \textbf{heterogeneity collapse}. Traditional dynamic routers compute merging coefficients for a given batch, average them across the batch, and reconstruct a single merged dense weight matrix. This design introduces three major failure modes:
\begin{enumerate}
    \item \textbf{I.I.D. Violation and Prediction Shifting:} Because weights are reconstructed using batch-averaged coefficients, a sample's prediction shifts dynamically depending on what other samples are co-packaged in the same batch, violating safety and reliability standards.
    \item \textbf{Heterogeneity Collapse:} In mixed-task streams, batch averaging forces the router to average opposing task-expert coefficients, producing an intermediate, uniform-like weight state that performs poorly for all samples (see Figure~\ref{fig:heterogeneity_collapse_intro}).
    \item \textbf{Compute and Memory Overhead:} Reconstructing massive, dense weight matrices at runtime introduces substantial latency and memory bandwidth bottlenecks, which is completely unviable for resource-constrained edge CPUs and NPUs.
\end{enumerate}

To resolve these critical bottlenecks, we propose \textbf{Sparse Low-Rank Dynamic Merging (SLD-Merge)}, a batch-independent, parameter-efficient, and computationally lean dynamic model-merging framework designed for robust real-world multi-task deployment. 
SLD-Merge shifts the dynamic adaptation from the heavy parameter space to a lightweight, sample-wise activation space. We extract the parameter task vectors of specialized experts and perform offline Singular Value Decomposition (SVD) once, decomposing them into high-fidelity, low-rank matrices of rank $r \ll D$ (e.g., $r=8$). 

During the forward pass, we employ a bounded cosine-similarity router that extracts global pooled representation vectors from input activations and computes sample-wise alignment scores. We enforce Top-1 sparse routing (hard gating) to activate only the single most relevant low-rank expert adapter for each individual sample. Since this process is executed via a vectorized, sample-wise parallel PyTorch forward pass, each input sample is processed completely independently, with zero cross-sample leakage and zero batch-size dependency.

Furthermore, we introduce \textbf{Activation-Space Mean Initialization}. We show that by simply setting the routing basis vectors to the average activation representing each task on a tiny unlabeled calibration set, SLD-Merge achieves peak performance completely \emph{zero-shot}, eliminating the need for labeled gradient descent calibration.

Our contributions are summarized as follows:
\begin{itemize}
    \item We characterize and empirically expose the ``heterogeneity collapse'' and batch-dependency of existing dynamic merging methods under varying batch sizes and shuffled mixed-task streams.
    \item We present Sparse Low-Rank Dynamic Merging (SLD-Merge), a framework combining offline SVD task-vector decomposition, bounded cosine-similarity routing, and sample-level hard Top-1 gating to achieve stateless, deterministic, batch-independent inference.
    \item We introduce Activation-Space Mean Initialization, demonstrating that highly competitive dynamic routing can be achieved zero-shot without any backpropagation calibration.
    \item Through extensive evaluation on a 4-dataset vision transformer benchmark (MNIST, FashionMNIST, CIFAR-10, SVHN), we demonstrate that SLD-Merge maintains a stable, peak joint accuracy of \textbf{63.87\%} across all batch sizes, completely eliminating heterogeneity collapse and outperforming state-of-the-art dynamic baselines by up to \textbf{+8.50\%} while reducing storage overhead by over \textbf{92.5\%} and adding only \textbf{8.3\%} FLOPs.
\end{itemize}
